**Тестировщик:** Ilia Abramov  
**Тестовое окружение:** Windows 11, Google Chrome 126.0  

---

**Номер:** BUG-1  
**Название:** Корзина: кнопка "Удалить" долго удаляет промокод  
**Статус:** Новый  
**Приоритет:** Низкий  

**Шаги для воспроизведения:**  
1. Добавить в корзину товар по ссылке https://lazurit.com/...
2. Перейти в корзину
3. Ввести валидный промокод в поле ввода и нажать "Применить"
4. Нажать на кнопку "Удалить" (крестик) рядом с применённым промокодом

**Фактический результат:** Промокод удаляется с существенной задержкой (более 3-5 секунд), при этом нет индикатора загрузки.  
**Ожидаемый результат:** Промокод удаляется мгновенно или отображается индикатор загрузки (лоадер).
