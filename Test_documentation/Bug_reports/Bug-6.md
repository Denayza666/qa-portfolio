**Номер:** BUG-6
**Название:** Форма регистрации: поле "Имя" принимает пробелы как валидное значение
**Статус:** Новый
**Приоритет:** Низкий

**Шаги для воспроизведения:**
1. Открыть форму регистрации нового пользователя
2. Заполнить все обязательные поля корректными данными, кроме поля "Имя"
3. В поле "Имя" ввести несколько пробелов (например, "   ")
4. Нажать кнопку "Зарегистрироваться"

**Фактический результат:** Форма успешно отправляется, создаётся аккаунт пользователя, у которого имя состоит только из пробелов.
**Ожидаемый результат:** Под полем "Имя" появляется сообщение об ошибке валидации (например, "Поле Имя не может состоять только из пробелов"), форма не отправляется.
