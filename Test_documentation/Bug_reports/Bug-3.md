**Тестовое окружение:** Windows 11, Google Chrome 126.0

**Номер:** BUG-3

**Название:** Избранное: счётчик иконки не отображает актуальное количество товаров при переходе между страницами

**Статус:** Новый

**Приоритет:** Средний

**Предусловие:** Корзина и избранное пусты. Пользователь авторизован.

**Шаги для воспроизведения:**
1. Открыть страницу товара (например, диван)
2. Нажать на иконку "Добавить в избранное" (сердечко)
3. Убедиться, что счётчик у иконки избранного в шапке сайта изменился на "1"
4. Перейти на любую другую страницу сайта (например, на главную)
5. Обратить внимание на счётчик избранного в шапке

**Фактический результат:** Счётчик избранного сбрасывается и не отображает добавленный товар.
**Ожидаемый результат:** Счётчик избранного сохраняет значение "1" и корректно отображает количество товаров при навигации по сайту.
