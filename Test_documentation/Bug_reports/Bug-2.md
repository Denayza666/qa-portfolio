**Номер:** BUG-2
**Название:** Корзина: при быстром удалении нескольких товаров с переходом на другую страницу удаляется только 1 товар
**Статус:** Новый
**Приоритет:** Низкий
**Воспроизводимость:** Зависит от скорости действий

**Шаги для воспроизведения:**
1. Добавить в корзину 4 любых товара
2. Открыть корзину
3. Нажать кнопку "Удалить" у 3 товаров быстро, без ожидания анимации удаления
4. Не дожидаясь завершения обновления корзины, перейти на главную страницу сайта
5. Вернуться в корзину

**Фактический результат:** В корзине удалился только 1 товар (первый из трёх).
**Ожидаемый результат:** Все 3 товара должны быть удалены из корзины.
