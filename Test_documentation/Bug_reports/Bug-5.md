**Тестовое окружение:** Windows 11, Google Chrome 126.0

**Номер:** BUG-5

**Название:** Корзина: при быстром переходе на другую страницу во время удаления товаров отображается ошибка 502 Bad Gateway

**Статус:** Новый

**Приоритет:** Высокий

**Воспроизводимость:** Плавающая, зависит от нагрузки на сайт

**Шаги для воспроизведения:**
1. Добавить несколько товаров в корзину
2. Перейти на страницу корзины
3. Быстро нажать кнопку "Удалить" у нескольких товаров подряд
4. Сразу же, до завершения анимации и запросов, кликнуть по ссылке на другую категорию в меню сайта

**Фактический результат:** Страница зависает, после чего отображается стандартная страница ошибки сервера 502 Bad Gateway.

**Ожидаемый результат:** Сервер корректно обрабатывает запросы, переход на другую страницу осуществляется без ошибок 5xx.
