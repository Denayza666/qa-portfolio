**Тестовое окружение:** Windows 11, Google Chrome 126.0

**Номер:** BUG-4
**Название:** Каталог "Кровати": при выборе всех значений фильтра "Подъёмный механизм" отображается меньше товаров, чем есть в категории
**Статус:** Новый
**Приоритет:** Низкий

**Шаги для воспроизведения:**
1. Открыть категорию «Кровати» (https://lazurit.com/catalog/648/krovati/)
2. Запомнить общее количество товаров в категории (без фильтров)
3. В блоке фильтров раскрыть "Подъёмный механизм"
4. Отметить все доступные чекбоксы в этом фильтре (например, "Да" и "Нет")
5. Сравнить полученное количество товаров с изначальным

**Фактический результат:** При выборе всех вариантов фильтра отображается меньше товаров, чем общее количество товаров в категории (некоторые товары не попадают ни в одну выборку фильтра).
**Ожидаемый результат:** При выборе всех исчерпывающих вариантов фильтра количество отображаемых товаров должно совпадать с общим количеством товаров в категории.
